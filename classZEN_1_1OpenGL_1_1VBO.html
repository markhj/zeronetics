◆ allocate()

std::optional< ZEN::GPUAllocation > ZEN::OpenGL::VBO::allocate ( ZEN::gpu_alloc_int  size)

Get an allocation block for X floats.

Note
You allocate spots for floats, not vertices. This means that if you want to allocate room for 3 vertices with 2 properties with a size 6, you must request a block of 18 (3x6).
When a std::nullopt was returned, it means there wasn't space in the VBO, and you will have to resize it.
The reason this method doesn't automatically resize the buffer when needed, is that it will clear the buffer data, and this has to be managed elsewhere.
