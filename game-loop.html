Zeronetics Game Engine
Game loop

The game loop, also often referred to as render loop, handles all the logic as the in-game is running. Anything from calculating enemy health to rendering fog.

üîÅ How it works

The game loop in Zeronetics is managed by ZEN::Game::run.

For each iteration, it will:

  • Clear render buffers
  • Process requests by entities
  • Render the scene
  • Handle user inputs
  • Swap render buffers

In between it also calculates FPS, and true render time.
