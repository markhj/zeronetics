Install and build engine

The first step toward building a game with Zeronetics is to install and build the engine.

The engine will be set up in a neutral location, and then referenced by your game using an environment variable. But more about that in Create a game.

There's not a single way to build the engine because this depends on your IDE or whether you use plain cmake.

Important: Preferably, you build the engine for both debug and release mode. That will make easier for you, as you're probably going to switch between the two when developing your game.
