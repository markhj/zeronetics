Detailed Description

Settings on a per-layer basis. Every layer is coupled with an instance of this struct, which allows every layer to have individual preferences such as depth testing and draw mode.

◆ depthTesting

bool ZEN::RenderSettings::depthTesting = true

Enable or disable depth testing

When depth resting is enabled it's the Z-coordinate of a vertex which determines the visible order. When it's disabled, it's the order in which vertices are rendered which take priority.
